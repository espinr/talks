Brief history...

Open Government Data

  • 2009 W3C Open Government Data paper!!!!
  • 2010-2015 Open data gaining traction in the public sector
  • Data in sports was is usually open
    • Event info: venue, schedule, prices, sports discipline,...
    • Athlete profiles: name, nationality, picture, DoB...
    • In-competition: heats, bib numbers, trials, partial results...
    • Post-competition: results, amendments...
    • Stats: records, rankings, ...

Notes.

Brief history...

OpenTrack CG

  • OpenTrack, an innovative EU project to apply `open data` in athletics.
  • EA and its ~60 member federations have ~60 ICT systems and formats.
  • We all wanted interoperability and federation of datasets.
  • We launched the OpenTrack CG at IODC in Madrid

Open Athletics

We use this name to avoid missunderstandings.

Athletics models and vocabularies

  • Ontology for Athletics competitions: too complex, so we simplified it
  • Abstract model for Athletic competitions and athletes
  • RDF vocabulary with examples in JSON-LD.
  • Taxonomies for: event types, categories, type of venues,...

Achievements and current status

  • Practical open source tool with reference APIs (used by XX federations/clubs)
  • European Athletics calendar uses the APIs. However, not fully federated yet.
  • World Athletics (top governing body) has released its API for a centralized calendar.
  • A lot of nice products and services deployed in specific countries.
