(e-books) EPUB

  • Content documents
    • XHTML profile with deviations, restrictions and extension
    • SVG profile
  • Other resources: CSS profile, images, videos,...
  • Allows (limited) scripting

(TV) Media Profiles

  • (2014, obsolete) CSS TV Profile 1.0
    • CSS WG does not publish CSS profiles since 2018
    • Now tv included as a media type
  • (2012, ED) Web Media Profile
  • (2018, CG Report) Web Media Application Developer Guidelines
  • (2024, CG Report) Web Media API for writing portable media web apps across browsers on consumer products (e.g., smart TVs, game machines, set-top boxes)

Non-standard attempt: AMP

AMP HTML is a subset of HTML for authoring content pages such as news articles in a way that guarantees certain baseline performance characteristics.

HTML, CSS minimized, JS limited, pre-fetching, caching...
