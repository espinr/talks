Case Study: MiniApps

  • CSS: subset of CSS3 (+a few additions)
  • Scripting: ECMAScript with limited execution
  • Markup: ~HTMLish domain-specific language
  • APIs: platform-specific, but similar functionality of Web APIs

See the standards gap analysis of MiniApp APIs and MiniApp elements.
