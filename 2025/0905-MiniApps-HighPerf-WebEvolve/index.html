Web Application Profiles?

  • Support development of dedicated embedding browsers
  • Guarantee interoperability within use cases
  • Maximize engine performance in concrete use cases

How to Develop Profiles?

  • Reference guidelines (for web engine developers) profiles of standards
  • "Web Features" to map profiles (MUST/SHOULD/MAY-Feature-profile)
  • Extending "Baseline" for "Web Profiles" (e.g., Shopping MiniApp Web Profile, AI Chatbot Web Profile)
  • Creating specific documents in Open Web Docs
  • Extending CanIUse (CanIWebProfileUse)? for integration with toolsets (e.g., editors, frameworks)
