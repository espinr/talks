XML Query: Éxito y Futuro

XML Query es el puente entre los documentos con una estructura interna compleja e irregular, y las bases de datos con valores simples y atómicos.

  • Recomendaciones en enero de 2007
  • XQuery 1.0: 40 implementaciones y 14.000 casos de prueba

    XQuery will serve as a unifying interface for access to XML data, much as SQL has done for relational data
    — Don Chamberlin, IBM, co-inventor del SQL

  • XSLT 2.0
    • Soporte para XML Schema, más funciones, facilidad para hacer agrupaciones, procesamiento de texto potente usando expresiones regulares.
  • XQuery 1.0 y XSLT 2.0 construidos sobre XPath 2.0

Próximos pasos sobre XML

  • XQuery 1.0 y XPath 2.0 Extensión de Texto-Completo para encontrar frases y palabras en XML

  • XQuery 1.1: Se construye sobre implementaciones y experiencia de los usuarios

Grupo de Trabajo de HTML

  • "HTML clásico"
  • Primer Borrador de Trabajo Público de HTML para junio de 2007
  • Widgets de Interfaces de usuario: barras de progreso, rejillas de datos, menús, y otros controles de formularios
  • Compatibilidad "hacia atrás" con los navegadores HTML
  • Características WYSIWYG en la edición

Grupo de Trabajo de XHTML2

  • Centrado en: i18n, accesibilidad, independencia de dispositivo y estructura de documento
  • Pieza esencial para enriquecer el contenido de los documentos con otras áreas del W3C
    • Matemáticas, SVG, Multimedia sincronizada, y formularios

  • XHTML-Print (Recomendación en septiembre de 2006)
  • RDFa
  • XHTML2 Incluirá XForms para su módulo de Formularios
