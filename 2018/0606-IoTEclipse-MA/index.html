The context: Athletics

The most popular sport

Most followed discipline at Summer Olympic Games

  • USA: 30,000+ running events; 47 million runners
  • Denmark: 31% runners
  • Germany: 25% runners

Source: Statista (2017); and Running across Europe The Rise and Size of one of the Largest Sport Markets (2015)

First-class running events

Fully Automatic Timekeeping: video, RFID,…

  • Reliable, real-time results, easy for organizers
  • Expensive solution; no standard exports

My proposal

Timekeeping system for races

  • Low cost
  • Device independent
  • Plug-and-play
  • Open Data compliant (open, standard formats)

OpenTrack

Vocabulary to describe Athletics competitions

  • Based on schema.org and JSON-LD
  • Under development by W3C Opentrack Community Group
  • Objective: create standards to describe and publish Athletics data (calendars, results, competitors, etc.)
